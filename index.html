Welcome

Welcome to the About Me Website project! This repository contains the collaborative work of 40 students from two different AP Computer Science Principles (APCSP) classes at Noble Street College Prep.

Project Overview

This project aims to develop an engaging and informative personal website for each student, showcasing their unique interests, skills, and aspirations.
